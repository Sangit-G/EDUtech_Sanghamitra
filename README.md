USE CASE OF AI EDUTECH:
This folder contains a small AI-powered EdTech platform built with FastAPI on the backend and a simple HTML frontend.
Its use case is to help students learn by generating personalized learning paths, quizzes, and explanations using the OpenAI GPT-4 API. 
The system stores student data, progress, and quiz results in an in-memory database and exposes multiple API endpoints for student management and AI-generated content. 
The main technologies used are FastAPI, Python, OpenAI API, dotenv, and a lightweight custom frontend.
